# NestTunes 🎵

A modern JavaFX music player application with playlist management and audio visualization.


## Features ✨
- 🎶 Play MP3/WAV audio files
- 📂 Create and manage playlists
- 🔍 Search through your music library
- 🎨 Sleek JavaFX UI with dark/light themes
- 📊 Audio visualization (waveform/spectrum)

## Prerequisites 📋
- Java JDK 17+
- JavaFX 17+ SDK
- Maven (for building)
